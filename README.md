# Generative AI Course Platform

A comprehensive learning platform for Generative AI, designed with a GeeksforGeeks-inspired interface.

## Features

- **GFG-inspired Design**: Clean, professional layout with green color scheme
- **Structured Learning**: Organized course modules with clear progression
- **Interactive Navigation**: Top header, left sidebar, and breadcrumb navigation
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Modern UI**: Card-based layouts, smooth animations, and professional typography

## Course Modules

1. **Introduction to AI** - Fundamentals of Artificial Intelligence
2. **LLM (Large Language Models)** - Deep dive into language models
3. **Prompt Engineering** - Master effective prompting techniques
4. **Creating AI Agents** - Build intelligent autonomous agents
5. **AI Tools & Frameworks** - Essential tools and platforms for AI development
6. **Projects** - Hands-on implementation with real-world applications

## Technology Stack

- **Frontend**: React 18 with React Router
- **Styling**: Pure CSS with modern design principles
- **Build Tool**: Vite for fast development
- **Icons**: Unicode emojis for visual elements

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```

3. **Open in Browser**
   Navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
